# WonMarket
## 팀 프로젝트 명 
WonMarket<br>

## 소개<br>
간단한 의류 쇼핑몰로 주문, 결제, 회원, 관리자, 리뷰, 게시판 등 의 기능이 들어가있습니다.<br>

## 프로젝트 기간 
22/06/15 ~ 22/07/15<br>

## 팀원 
이용학, 최원진, 임병기, 최종안, 최경준, 주동현, 지경대<br>
